Datatype & Data structure 


numeric : int , float complex
Boolean : True/False
sequence : list, tuple, range()

Range() : It is a function. It is used to generate sequence of number.

range(s, s, s)

s : start (by default 0, when it is not mentioned)
s : stop (exclude)
s : step (by default 1, when it is not mentioned)


range(10) : 0.....9
range(1, 10) : 1....9
range(1, 10, 2) : 1, 3, 5, 7


100.... 50

a = range(100, 50, -1)
