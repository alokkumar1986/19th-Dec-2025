Data type :

The type that a variable holds in terms of value, is called as data type.

There are several data type

1) numeric : int, float, complex
2) string : str
3) Boolean : True, False
4) map : dict
5) set : set, frozen set
6) bytes : bytes array
7) sequence : list, tuple, range 
8) None : None


numeric datatype :

Any number comes under numeric datatype. 

If a number is a whole number with negative sign, it is called integer data type or int data type.

a = 10
b = 15
c = -100
print(type(a)) # <class 'int'>
print(type(b)) # <class 'int'>
print(type(c)) #<class 'int'>

If a number having negative sign and 1 or more decimal points, it is called float data type

a = 10.0
b = 15.12
c = -100.0
print(type(a)) # <class 'float'>
print(type(b)) # <class 'float'>
print(type(c)) #<class 'float'>


If a number is having a real part and an imaginary part, it is called complex data type

a = 1+10j
b = -2-5j
print(type(a)) #<class 'complex'>
print(type(b)) #<class 'complex'>


print(a.real)


Type Casting : Conversion of one datatype to other explicitly.

int() : To convert into int
float() : To convert into float
complex() : To convert into complex
